📚 BookNest

BookNest is a simple and elegant Flutter application that simulates an electronic library. The app allows users to browse available books, search for specific titles, and view detailed information about each book.

✨ Features
🚀 Splash Screen with the BookNest logo
📚 Home Screen displaying a list of books
🔍 Search functionality
📖 Book Details Screen
⭐ Book ratings display
🎨 Clean and modern UI
📱 Android compatible
📚 Available Books
Clean Code
Atomic Habits
Rich Dad Poor Dad
Deep Work
The Psychology of Money
📌 Book Information

Each book contains:

Title
Author
Cover Image
Short Description
Rating
🎨 Design
Minimal and user-friendly interface
White and Dark Blue color palette
Responsive layout for different screen sizes
